**Key Observations**:
- Every single CVE has at least one Patch reference
- Most CVEs have **6-8 patch references** (different kernel stable branches)
- Patches typically point to git.kernel.org stable branches
- Average of **8.7 references per CVE**

**Typical Patch Pattern**:
CVEs include patches for multiple stable kernel versions (e.g., 5.4.x, 5.10.x, 5.15.x, 6.1.x, 6.5.x), showing comprehensive backporting efforts.

## Summary Insights

1. **Severity Profile**: Evenly split between HIGH (45%) and MEDIUM (55%) severity
2. **Primary Threat**: Use-after-free vulnerabilities (CWE-416) are most severe
3. **Attack Surface**: Nearly all require local access (97%), limiting remote exploitation
4. **Version Impact**: Some vulnerabilities affect 10+ years of kernel versions
5. **Remediation**: 100% patch availability demonstrates strong kernel security response
6. **Risk Ratio**: High impact but lower exploitability favors defense-in-depth strategies

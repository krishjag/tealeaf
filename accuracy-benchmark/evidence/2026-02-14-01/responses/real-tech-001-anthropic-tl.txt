## 4. Attack Vector Analysis

| Attack Vector | Count | Percentage |
|---------------|-------|------------|
| **LOCAL** | 38 | 95.0% |
| **NETWORK** | 2 | 5.0% |
| **ADJACENT_NETWORK** | 0 | 0.0% |
| **PHYSICAL** | 0 | 0.0% |

**Key Finding:** **95% of vulnerabilities require local access**, meaning attackers need authenticated access or code execution on the target system. Only 2 CVEs (CVE-2021-47075, CVE-2023-52511) are exploitable over the network, significantly reducing the remote attack surface.

## 5. Broadest Kernel Version Range Affected

**CVE-2022-48627** (VT memory corruption vulnerability) affects the **broadest range**:
- **Span:** Linux kernel 3.7 through 6.6-rc6
- **Range:** Approximately **10+ years** of kernel versions (2012-2023)
- Affected versions: 3.7→4.19.312, 4.20→5.4.274, 5.5→5.10.132, 5.11→5.15.56, 5.16→5.18.13, plus 5.19 release candidates

**Runner-up:**
- **CVE-2023-52502** (NFC LLCP race condition): 3.6 through 6.6-rc5
- **CVE-2023-52522** (neighbor cache tearing): 2.6.37 through 6.6-rc5 (longest historical reach)

## 6. Exploitability vs Impact Analysis

**Average Exploitability Score:** 1.83  
**Average Impact Score:** 4.34  
**Impact to Exploitability Ratio:** 2.37:1

**Analysis:** Impact scores are on average **2.4 times higher** than exploitability scores, indicating these vulnerabilities cause significant damage when exploited but are moderately difficult to exploit (primarily requiring local access with low privileges).

### Score Breakdown:
- **Exploitability Range:** 1.0 - 3.9
- **Impact Range:** 1.4 - 5.9
- **Highest Exploitability:** CVE-2023-52511 (3.9 - network accessible)
- **Highest Impact:** CVE-2021-47069, CVE-2021-47081, CVE-2023-52503, CVE-2023-52504, others (5.9)

## Summary Insights

1. **Risk Profile:** Predominantly local exploitation requiring authenticated access, with medium-to-high impact potential
2. **Memory Safety:** 50% of classified vulnerabilities involve memory corruption issues (UAF, OOB, NULL deref, memory leaks)
3. **Patch Availability:** Excellent - 100% have upstream patches available across multiple kernel versions
4. **Legacy Impact:** Several vulnerabilities affect kernel versions spanning 5-10 years, indicating long-standing code issues
5. **Concurrency Issues:** Notable presence of race conditions (CWE-362) and improper locking (CWE-667), typical of kernel development challenges
